# SwitchCast App Icon 设计方案

## 产品定位回顾

SwitchCast = **Switch** + **Cast**（投屏/投射）

核心关键词：投屏、简单、即插即用、纯净
不是：直播工具、游戏模拟器、视频编辑器

Icon 需要传达的信息层级：
1. 第一眼：这是一个和"屏幕/投屏"相关的工具
2. 第二眼：简洁、现代、可信（开源软件）
3. 第三眼：暗示 Switch 到电脑的方向感

Icon 设计约束：
- 16x16 像素时仍可辨认（核心轮廓要够简单）
- 不依赖文字（缩小时看不清字）
- 不依赖细节（渐变、光影在小尺寸下会糊掉）
- 与系统原生应用图标风格协调（macOS/Windows）

---

## 设计方案

### 方案 A：屏幕+投射线（推荐）

**设计理念**：一个圆角矩形代表"屏幕"，一条对角线从屏幕中射出代表"投射"。整个图标就是"把画面投出去"这一动作的极简符号化。不出现 Switch 主机、不出现手柄、不出现波纹——那些太具象了，缩到 32px 全糊。这个方案只用两个几何元素：一个矩形+一条线，16px 也能认。

**Prompt**：
```
一个桌面应用图标的极简设计，白色背景。图标主体为一个深蓝紫色圆角矩形（代表屏幕），从矩形右下角射出一条青色对角线条（代表画面投射出去）。矩形内部留白，线条从矩形边缘开始向外延伸，末端略微变细，像一束光投射出去的感觉。整体只有两个元素：一个矩形和一条线。配色为深蓝紫(#3B2A6B)矩形+青色(#00D4FF)线条。扁平化设计，无阴影，无渐变，无背景纹理。纯白底，中心构图，四周边缘留白充足。风格参考 macOS 系统应用图标的克制美学。
```

---

### 方案 B：点+弧线（信号投射符号）

**设计理念**：借鉴 AirPlay / Chromecast 的视觉语言——一个圆点（信号源）+ 弧形线（投射方向），但做差异化处理。圆点代表 Switch，弧线代表画面投射到另一块屏幕。比方案 A 多一点"连接"的语义，但仍然极度克制。

**Prompt**：
```
一个桌面应用图标的极简设计，白色背景。图标主体为一个实心圆点（位于左下，代表信号源）和两道向外扩散的弧形线条（代表画面投射方向），弧线从圆点右侧向外弯曲延伸。圆点为深蓝紫色(#3B2A6B)，弧线为青色(#00D4FF)。整体形态类似一个抽象的"广播"或"投射"符号，但不使用完整的同心圆，而是开口式的弧线，暗示"投射到另一处"。两个元素之间有留白，不相连。扁平化设计，无阴影，无渐变。纯白底，中心构图。
```

---

### 方案 C：双屏叠影

**设计理念**：两个错位重叠的圆角矩形，代表"源屏幕"和"目标屏幕"——Switch 的画面投射到了另一块屏幕上。后景矩形深色、前景矩形亮色，形成层次感。这个方案的语义是"两个屏幕"而不是"投射动作"，更偏向静态的产品属性表达。

**Prompt**：
```
一个桌面应用图标的极简设计，白色背景。图标主体为两个圆角方形部分重叠：后方的方形为深蓝紫色(#3B2A6B)，前方偏右下的方形为青色(#00D4FF)，两个方形有约30%的重叠区域。后方形代表源设备屏幕，前方形代表目标显示屏幕，重叠暗示"画面从一处投射到另一处"。两个方形均为实色填充，无描边，无阴影。扁平化设计，纯白底，中心构图，四周边缘留白充足。
```

---

### 方案 D：字母 S 变体

**设计理念**：取 SwitchCast 首字母 S，将其几何化为一条从左下到右上的流动曲线，曲线本身形成 S 形的同时也暗示"画面流动/投射"的方向感。S 的右下端可以略微断开或变细，暗示"投射出去"。

**Prompt**：
```
一个桌面应用图标的极简设计，白色背景。图标主体为一个大写字母S的几何化变体：S由一条粗细均匀的曲线构成，从左下角流向右上角，曲线在右下端略微延伸并变细，像画面投射出去的轨迹。字母S同时暗示SwitchCast的首字母和画面流动的方向感。配色为深蓝紫(#3B2A6B)到青色(#00D4FF)的渐变填充曲线。扁平化设计，无阴影，无背景装饰。纯白底，中心构图，字母占据图标面积的60%左右。
```

## 方案对比

| 维度 | A 屏幕+投射线 | B 点+弧线 | C 双屏叠影 | D 字母S |
|------|:---:|:---:|:---:|:---:|
| 16px 可辨认度 | 高 | 中 | 中 | 低 |
| 语义清晰度 | 高（投屏动作） | 高（信号源+方向） | 中（两个屏幕） | 低（需识别字母） |
| 与竞品区分度 | 高 | 中（像AirPlay） | 高 | 高 |
| 品牌记忆点 | 中 | 中 | 低 | 高（字母记忆） |
| 推荐度 | 首选 | 备选 | 可选 | 不推荐首版 |

**推荐方案 A**：屏幕+投射线。原因：
- 16px 仍可辨认（矩形+对角线 = 2 个几何元素）
- 语义直接（"把屏幕里的东西投出去"）
- 不像任何竞品（AirPlay 是同心弧，OBS 是三色圆环）
- 克制、现代、系统级应用气质

## 尺寸规格

确定方案后，生成以下尺寸：

| 尺寸 | 用途 |
|------|------|
| 1024x1024 | App Store / electron-builder icon 来源 |
| 512x512 | macOS .icns 中尺寸 |
| 256x256 | Windows .ico 中尺寸 |
| 128x128 | 小尺寸 |
| 64x64 | 最小展示 |
| 32x32 | 任务栏/Dock |
| 16x16 | 系统托盘 |

生成 1024x1024 主图后，可手动裁剪缩放其他尺寸。最终需要转换为 `.icns`（macOS）和 `.ico`（Windows）格式放入 `build/` 目录。
